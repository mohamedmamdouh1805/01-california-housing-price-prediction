# California Housing Price Prediction

## Overview
Machine Learning project to predict housing prices using the California Housing Dataset.

## Problem Type
Regression Problem

## Dataset
California Housing Dataset from Scikit-Learn.

## Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Preprocessing

- Checking missing values
- Train/Test split
- Feature Scaling using StandardScaler

## Evaluation Metrics

- MAE
- RMSE
- R² Score

## Results

| Model | R² Score |
|---|---|
| Linear Regression | 0.576 |
| Decision Tree | 0.623 |
| Random Forest | 0.805 |

## Best Model

Random Forest achieved the best performance.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook
